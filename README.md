# Nex-Player
Simple Music player written in Python

## Dependencies required to run this.

* ttkthemes
* stagger
* pygame
* tinytag
* mutagen
* Pillow

## Installing the dependencies
```
pip install -r requirements.txt
```

## Running the project
```
git clone https://github.com/Mr-sud0/Nex-Player
cd Nex-Player
python music.py
```
